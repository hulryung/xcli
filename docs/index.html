Post to X from your terminal.

A fast, lightweight CLI for X (Twitter). Tweet, thread, and delete — all from your command line. Built with Rust.

Built for Speed & Simplicity

Everything you need to post on X without leaving your terminal.

edit_note

One-Line Tweeting

xcli tweet "Hello!" — that's it. Compose and post in a single command.

call_split

Auto-Threading

Long text is automatically split into threads at 280 chars, or use --- to control split points. Preview with --dry-run.

group_add

Team OAuth

Admin shares API Key/Secret, each member authenticates with their own X account via browser-based OAuth flow.

public

Cross-Platform

Native binaries for macOS, Linux, and Windows. Install via Homebrew, Winget, Cargo, or download directly.

Simple Commands

Intuitive subcommands that do exactly what you'd expect. No complex flags, no configuration files to learn.

examples.sh 
# Post a tweet
$ xcli tweet "Hello from xcli!"
Tweet posted! ID: 1893847562031

# Post a thread with manual splits
$ xcli tweet "First tweet --- Second tweet --- Third tweet"
Thread posted! (3 tweets)

# Preview thread without posting
$ xcli tweet "Long text here..." --dry-run
Thread preview (2 tweets):
  [1/2] (250/280) First chunk...
  [2/2] (180/280) Second chunk...

# Delete a tweet
$ xcli delete 1893847562031
Tweet 1893847562031 deleted.

# OAuth login (opens browser)
$ xcli auth login
Logged in as @username
